# Show-ColorsWithBackground

## SYNOPSIS
Show-ColorsWithBackground displays all combinations of foreground/background of the console colors.

## SYNTAX

```
Show-ColorsWithBackground [<CommonParameters>]
```

## DESCRIPTION
Show-ColorsWithBackground displays all combinations of foreground/background of the console colors.
