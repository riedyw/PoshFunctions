Get-ScheduledTaskUser

SYNOPSIS

Get a list of scheduled tasks and the user context that they run under. By default excluded system tasks

SYNTAX

Get-ScheduledTaskUser [[-ComputerName] <String[]>] [[-UserName] <String>] [-IncludeSystem]
 [-IncludeInteractive] [<CommonParameters>]

DESCRIPTION

Get a list of scheduled tasks and the user context that they run under. By default excluded system tasks
