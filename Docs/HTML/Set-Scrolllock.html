Set-Scrolllock

SYNOPSIS

Sets the state of the ScrollLock button.

SYNTAX

On (Default)

Set-Scrolllock [-On] [-WhatIf] [-Confirm] [<CommonParameters>]

Off

Set-Scrolllock [-Off] [-WhatIf] [-Confirm] [<CommonParameters>]

DESCRIPTION

Sets the state of the ScrollLock button. It first determines the state of the ScrollLock and then acts accordingly.

EXAMPLES

EXAMPLE 1

Set-ScrollLock

Will turn on the ScrollLock

EXAMPLE 2

Set-ScrollLock -Off

Will turn off the ScrollLock
