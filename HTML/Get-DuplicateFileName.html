Get-DuplicateFileName

SYNOPSIS

To find duplicate file names within a given folder

SYNTAX

Get-DuplicateFileName [[-Path] <String>] [<CommonParameters>]

DESCRIPTION

To find duplicate file names within a given folder. Alias for function 'Get-DupeFileName'. Will run Get-ChildItem against the given path, groups them by filename, and finds those that have a count greater than 1
