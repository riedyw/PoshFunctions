Get-Shortcut

SYNOPSIS

Get information about a Shortcut (.lnk file)

SYNTAX

Get-Shortcut [[-Path] <String>] [<CommonParameters>]

DESCRIPTION

Get information about a Shortcut (.lnk file)

EXAMPLES

EXAMPLE 1

Get-Shortcut -path C:\portable\test2.lnk

LinkPath : C:\portable\test2.lnk Link : test2.lnk TargetPath : C:\Windows\System32\ncpa.cpl Target : ncpa.cpl Arguments : Hotkey : WindowStyle : Normal IconLocation : %SystemRoot%\system32\ncpa.cpl,0 RunAsAdmin : False

PARAMETERS

-Path

Path to .lnk file

Type: String
Parameter Sets: (All)
